# generate-flag-avatar
不需要@微信官方，迎国庆换新颜活动，给头像添加一面国旗

### 演示地址
<img src="/src/qr.png" width="200" height="200" />

### 使用说明
本机需要 go 环境

把要生成的头像放在根目录下,修改`main.go`里的头像路径

运行 `main.go`,四种结果在`output`文件夹内
